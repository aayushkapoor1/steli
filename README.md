# Steli

**Steli** is an Android app that enables users to share their favorite study spots around campus. Users can add reviews for study spots, rank them against one another, and share their ranked lists with friends.

## Project structure

- **`android/`** – Kotlin Android app (workshop-style Empty Activity with Jetpack Compose). See [android/README.md](android/README.md).
- **`backend/`** – Python FastAPI backend (optional). See [backend/README.md](backend/README.md).

### Quick start (Android)

Open the **`android/`** folder in **Android Studio**, run **File → Sync Project with Gradle Files**, then run the app on an emulator or device.
